# Small Office Network

## 📖 Project Overview

This project simulates a small office network using Cisco Packet Tracer.

## 🏢 Company Scenario

A small company has three departments:

- HR
- IT
- Finance

The goal was to build a basic Layer 2 network and verify communication between devices.

## 🛠️ Tools Used

- Cisco Packet Tracer
- Cisco Catalyst 2960 Switch
- Cisco CLI

## ✅ Features

- Connected three departments to a Layer 2 switch
- Configured IP addresses
- Tested connectivity using Ping
- Verified the MAC Address Table
- Configured basic Port Security

## 📚 Skills Learned

- Layer 2 Switching
- MAC Address Learning
- Cisco CLI
- Port Security
- Network Connectivity Testing
